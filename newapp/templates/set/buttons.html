🔘 Interactive Buttons

Create button groups that the AI can send as interactive WhatsApp messages. Users tap buttons to trigger actions or navigate to URLs.

💡 How to Use

Add the button tag to your AI Agent prompt to tell it when to show buttons:

{% templatetag openvariable %}button:group_name{% templatetag closevariable %}

Example prompt instruction: "When the user asks about our services, show them options using {% templatetag openvariable %}button:main_menu{% templatetag closevariable %}"

WhatsApp Limits: Max 3 buttons per group. Button titles max 20 characters.
