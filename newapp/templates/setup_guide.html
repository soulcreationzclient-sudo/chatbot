🚀 SpeedBot Setup Guide

Everything you need to set up and start using your AI-powered chatbot platform.

1 Getting Started
  1. Log in to your SpeedBot dashboard using the login URL, ID, and password provided by the SpeedBot team.
  2. Once logged in, you'll see the Inbox — the main dashboard.
💡 First things to do: Connect your WhatsApp number, add your OpenAI API key, and set up your AI system prompt.
2 Connecting Your WhatsApp Account
  1. Navigate to Settings → Integration.
  2. Click on Connect WhatsApp.
  3. You will be redirected to the Meta (Facebook) Embedded Signup flow.
  4. Follow the on-screen steps:
    • Log in with your Facebook account.
    • Select or create a WhatsApp Business Account.
    • Choose the phone number you want to connect.
    • Complete the OTP verification.
  5. Once completed, you'll be redirected back — your WhatsApp number is now connected!
⚠️ The phone number must not already be registered on the regular WhatsApp app. Make sure your Meta Business account is verified.
3 Adding Your OpenAI API Key

How to Get an OpenAI API Key:

  1. Go to platform.openai.com/signup and create an account.
  2. Navigate to API Keys → platform.openai.com/api-keys.
  3. Click "Create new secret key".
  4. Copy the key immediately — it won't be shown again.
  5. Add billing/credits at OpenAI Billing.

Adding the Key to SpeedBot:

  1. Go to Settings → Integration.
  2. Paste your API key in the OpenAI API Key field.
  3. Click Save.
⚠️ Make sure your OpenAI account has credits/billing enabled. Without credits, the chatbot cannot generate responses.
4 Selecting a GPT Model
  1. Go to Settings → Integration.
  2. Find the GPT Model dropdown.
  3. Select your preferred model and click Save.
ModelBest ForCost
gpt-4oComplex conversations (recommended)Higher

💡 Start with gpt-4o-mini for testing (lower cost), then switch to gpt-4o for production.
5 Writing Your AI Prompt (System Prompt)

The System Prompt tells the AI who it is, how to behave, and what to do. Go to Settings → ChatGPT Prompt.

Example Prompt:

You are a friendly and professional customer support assistant for [Your Company Name]. ## YOUR ROLE - Help customers with product inquiries, order status, and general questions. - Be polite, concise, and helpful at all times. ## COMPANY INFORMATION - Company: [Your Company Name] - Products: [List your products/services] - Working Hours: Monday to Friday, 9 AM – 6 PM ## CONVERSATION RULES 1. Always greet the customer warmly. 2. Keep responses concise — no more than 2-3 short paragraphs. 3. Never make up information. ## USING TAGS - When a customer asks about pricing → {{tag:add:PRICING_INQUIRY}} - When a customer makes a purchase → {{tag:add:CUSTOMER}} ## USING CUSTOM FIELDS - When you learn the customer's name → {{custom_field:customer_name:THE_NAME}} - When you learn their email → {{custom_field:customer_email:THE_EMAIL}} ## SENDING IMAGES - When a customer asks about products → {{image:product_catalog}} ## BOOKING APPOINTMENTS (if Calendly is set up) - When a customer wants to schedule → {{calendly:booking_link}} - NEVER generate your own booking URLs.

Key Syntax Reference:

Add Tag{{tag:add:TAG_NAME}}{{tag:add:NEW_LEAD}}
Remove Tag{{tag:remove:TAG_NAME}}{{tag:remove:NEW_LEAD}}
Save Custom Field{{custom_field:name:value}}{{custom_field:customer_name:John}}

⚠️ Tag names, custom field names, and image asset names must exactly match what you create in the dashboard (case-sensitive).
6 Creating Tags

Tags help you categorize and segment your contacts (e.g., "New Lead", "Customer", "VIP").

  1. Go to Settings → Tags.
  2. Click "Add New Tag".
  3. Enter the Tag Name (e.g., NEW_LEAD, CUSTOMER).
  4. Optionally set a Keyword for auto-tagging.
  5. Click Save.

In your prompt, instruct the AI: {{tag:add:PRICING_INQUIRY}}

Tags appear in the Inbox right panel and Contacts page for filtering.

7 Creating Custom Fields

Custom fields store specific data about each contact (name, email, product interest, etc.).

  1. Go to Settings → Custom Fields.
  2. Click "Add New Field".
  3. Enter the Field Name and select the Field Type.
  4. Click Save.

Available Field Types:

8 Setting Up Follow-Up Messages

Follow-ups automatically re-engage contacts who haven't replied.

  1. Go to Settings → Follow-Up Messages.
  2. Toggle Enable Follow-Ups to ON.
  3. Click "Add Follow-Up Step".
  4. Set the Message, Delay (minutes), and Active toggle.
  5. Click Save.

Example Sequence:

StepDelayMessage
110 minHi! 👋 Did you have any other questions?
260 minHey! Would you like more details about our products?
324 hrsHi again! 🌟 We're here whenever you're ready.
💡 If the user replies, all pending follow-ups are automatically cancelled.
9 Setting Up External APIs

External APIs let your chatbot fetch or send data to third-party services.

  1. Go to Settings → External APIs.
  2. Click "Add New API".
  3. Configure: Name, URL, Method (GET/POST), Headers, and Payload.
  4. Click Save.

In your prompt: {{api:check_order}}

10 Setting Up Calendly Integration

Connect Calendly:

  1. Go to Settings → Integration.
  2. Click "Connect Calendly" and log in.

Set Up Booking Links:

  1. Go to Settings → Calendly Booking Links.
  2. Click "Add Booking Link".
  3. Set: Name, URL, optional Custom Field, and Booking Message.
  4. Click Save.

In Your Prompt:

## APPOINTMENT BOOKING RULES 1. When a customer wants to book → ALWAYS use {{calendly:booking_link}} 2. NEVER ask for name, phone, email manually — Calendly handles it. 3. NEVER generate your own URLs — always use the tag.
💡 When a user books via Calendly, SpeedBot automatically updates their contact with booking status.
11 Uploading Image Assets

Image assets let your chatbot send images (product photos, menus, brochures).

  1. Go to Settings → Image Assets.
  2. Click "Upload Image".
  3. Enter the Asset Name (e.g., product_catalog).
  4. Select the image file and click Upload.

In your prompt: {{image:product_catalog}}

⚠️ Asset name must exactly match — use lowercase with underscores, no spaces.
12 Using the WebChat Widget

Embed a chatbot widget on your website for visitors to chat in real-time.

  1. Go to WebChat → Widgets.
  2. Click "Create Widget".
  3. Customize: name, color, greeting message, position.
  4. Click Save, then copy the embed code.
  5. Paste it into your website's HTML before </body>.
💡 WebChat uses the same AI prompt as WhatsApp — no separate setup needed!
13 Test Chat

Test your prompt before going live:

  1. Go to Test Chat in the sidebar.
  2. Type a message and press Send.
  3. Verify responses, tags, custom fields, images, and Calendly links work correctly.
💡 Test greetings, product inquiries, edge cases, and objections for a complete check.
